# Note App using Bloc

### Notes App is a simple application devloped by flutter 

## Tools used

- sqflite as local db
- Bloc(cubit) As State Management

## Main Features

- Dark/Light Mode
- Add new note
- Edit note
- Delete note
- Change the view of notes list
